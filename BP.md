# Ocean Canvas 深海画室 — 商业计划书（初级版）

## 一、项目概述

**一句话定义**：让用户手绘深海生物，AI识别评分后放入共享深海鱼缸观赏互动的Web应用。

**项目定位**：社区驱动型创作社交平台，介于 DrawAFish.com（单一画鱼）和 Skribbl.io（竞技画画）之间的新品类——**"创作+观赏+养鱼"**三位一体。

**核心差异**：
- DrawAFish 只能画鱼 → Ocean Canvas 支持8种深海生物
- DrawAFish 无AI识别 → Ocean Canvas 用 Cloudflare Workers AI 做实时识别+创意评分
- DrawAFish 无变现 → Ocean Canvas 有装饰商店+季票+广告三层变现
- DrawAFish 简陋UI → Ocean Canvas 深海赛博朋克沉浸式视觉

## 三、产品架构

### 3.1 核心功能模块

```
Ocean Canvas
├── 绘制引擎
│   ├── 8种深海生物模板（鱼/水母/章鱼/海龟/螃蟹/鲸鱼/鲨鱼/海马）
│   ├── 高级画笔（颜色/粗细/橡皮/撤销/快选色板）
│   └── AI实时评分（每笔停顿1.5秒自动识别）
│
├── AI识别系统（核心竞争壁垒）
│   ├── Cloudflare Workers AI（Llama 3.2 Vision）
│   ├── 相似度评分（≥60%通过，<30%弹窗引导）
│   ├── 创意评分（0-100分，考虑细节/色彩/构图）
│   └── 智能建议（识别为其他生物时提示切换类型）
│
├── 深海观赏（留存核心）
│   ├── 全屏动画鱼缸（光线/气泡/海草/深海渐变）
│   ├── 生物游泳动画（按类型差异化：水母脉动/鱼摇摆/螃蟹横行）
│   ├── 投放食物交互（引诱生物聚集）
│   └── 排序筛选（最新/最热/随机）
│
├── 排行投票（社交驱动）
│   ├── 4种排序（热榜/最高分/最新/随机）
│   ├── 投票系统（👍/👎）
│   └── AI创意分展示
│
└── 变现系统（Phase 2）
    ├── 装饰商店（水草/珊瑚/沉船/霓虹灯牌）
    ├── 深渊季票（限定生物+限定装饰）
    └── 非侵入式广告
```

### 3.2 技术架构

```
前端：纯HTML/CSS/JS + Canvas 2D
托管：Cloudflare Pages（全球300+ PoP，免费起步）
AI：Cloudflare Workers AI（@cf/meta/llama-3.2-11b-vision-instruct）
API：Cloudflare Pages Functions（/api/classify）
数据：localStorage（MVP期）→ Supabase（增长期）
部署：Git push → Cloudflare Pages 自动部署
```

**技术成本**：日活<1万时月成本 ¥0

## 四、AI识别系统 — 核心竞争壁垒

### 4.1 与DrawAFish的AI对比

| 维度 | DrawAFish | Ocean Canvas |
|------|-----------|-------------|
| AI模型 | 自训练ONNX模型（仅识别鱼） | Cloudflare Workers AI Vision（通用视觉理解） |
| 运行位置 | 浏览器本地（需下载模型） | 服务端边缘（零下载） |
| 识别范围 | 仅鱼 vs 非鱼 | 8种生物+创意评分+智能建议 |
| 模型更新 | 需重新训练+用户重新下载 | 服务端热更新，用户无感 |
| 扩展性 | 每加一种生物需重训模型 | 改prompt即可，无需重训 |
| 成本 | 零（本地推理） | Workers AI免费额度内零成本 |

### 4.2 AI识别流程

```
用户停笔1.5秒 → 自动发送画布到 /api/classify
                    ↓
         CF Workers AI Vision 分析图像
                    ↓
    ┌───────────────┼───────────────┐
    ↓               ↓               ↓
 相似度评分      创意评分        类型建议
 (0-100%)      (0-100分)     (可能识别为其他生物)
    ↓               ↓               ↓
  显示实时分数    显示创意分     "这更像一只水母？"
    ↓
 点击"放入深海"
    ↓
 相似度≥60% → 直接通过 ✅
 相似度<30% → 弹窗引导继续画 ⚠️
 30-60%之间 → 提交但标记待审核
```

### 4.3 竞争门槛分析

- DrawAFish的ONNX模型：竞品可直接复制模型文件
- 我们的Workers AI方案：**API在服务端，竞品看不到模型/prompt**
- 创意评分算法：服务端逻辑，**不可逆向**
- 即使竞品也用Workers AI，我们的**数据飞轮**（越多作品→识别越准→推荐越好）不可复制

## 十一、为什么是现在

1. **DrawAFish已验证需求但未建壁垒** — 用户正在流失，窗口期12-18个月
2. **Cloudflare Workers AI刚开放** — 零成本视觉AI能力，6个月前不存在
3. **AI绘画工具降低"画得好"的价值** — 反而提升"画得丑但有趣"的价值，这正是我们的生态位
4. **WebGPU成熟** — 深海动画效果1年前做不到
5. **国内品类空白** — "画鱼放鱼缸"在国内未火过，谁先做谁定义品类
